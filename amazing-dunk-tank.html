Project Information

  • Name: Amazing Dunk Tank
  • Category: 2D Puzzle Game
  • Platform: Windows
  • Engine: Unity
  • Project URL: saji002.itch.io/amazing-dunk-tank-update
  • Game Jam: WTF x IGDC Game Jam 2024 (48 hours)

A fun little mini-game where you modify the environment to make the ball hit the dunk tank!

Created for the 48-hour jam, this game challenges players to think strategically: use the ball, space and environmental elements to activate a button that triggers the dunk tank mechanism.

Development Highlights

This project demonstrates rapid-prototype game development under time constraints, creative problem-solving, and playful puzzle design.

  • Developed a 2D puzzle concept inspired by real-life dunk tanks and the mechanics of Amazing Alex.
  • Designed up to 6 distinct levels + a “Create Your Own Level” sandbox mode.
  • Rapidly iterated game mechanics and logic within 48 hours for the WTF x IGDC Game Jam 2024.
  • Released a Windows build (zip) to the community; simple install instructions included.
  • Focused on minimalist design and playful physics-driven interactions given jam constraints.
