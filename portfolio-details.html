Wobble Gobble - A Fun and Challenging Puzzle Adventure

"Wobble Gobble" is an isometric puzzle game where players control a jelly-like cube to navigate through challenging levels. The objective is to roll a separate cube onto specific targets to complete puzzles. The game features interactive mechanics like color-matching cubes, buttons, and teleporters, creating a unique and engaging experience. With three distinct worlds and multiple levels, players must think strategically and adapt to the dynamic physics-based gameplay. Available on PC and Android.

Development Contributions

I played a key role in the development of *Wobble Gobble*, focusing on various technical and artistic aspects of the game:

  • Implemented the "jelly effect", bringing the main cube to life with dynamic and responsive movement using C# and Unity's physics engine.
  • Designed and modeled the "first world" and other elements, creating an engaging visual and gameplay experience using Blender, Maya and Unity's modeling tools.
  • Worked on "level design", ensuring challenging yet enjoyable puzzles for players by crafting intricate level layouts and obstacles.
  • Developed "shaders", enhancing the game's aesthetics and visual feedback using Unity's ShaderLab and Cg programming language.
  • Implemented game logic using C# scripts, ensuring smooth gameplay and interaction with Custom Movement System.
  • Developed a dynamic camera system, allowing for a more immersive experience and adapting to the player's actions.
  • Created a user interface using Unity's UI system, including menus, buttons, and other interactive elements.
  • Optimized game performance by implementing efficient algorithms and reducing unnecessary computations.
  • Integrated sound effects and music using Unity's audio system, enhancing the overall gaming experience.
